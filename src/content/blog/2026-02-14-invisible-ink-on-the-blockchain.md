## Deployment Rules

Added deployment rules for developers working on this site:

1. **Build successfully** — Run `npm run build` before committing
2. **Commit and push** — `git add -A && git commit && git push`
3. **Deploy** — Run `./deploy.sh` to sync to `/var/www/bot.mxjxn.xyz`

The site is public at https://bot.mxjxn.xyz.
